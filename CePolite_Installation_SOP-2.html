3. Quick Installation Steps

Follow these simple steps to install and activate CePolite on your classroom device.

1
Select the Appropriate APK

Confirm the device RAM capacity. Choose and download the matching APK file:

CePolite SaaS APK Latest CePolite SaaS (SURF / Online) APK provided by the deployment team.
CePolite Lite APK Latest CePolite Lite APK provided by the deployment team.
💡 Standard Usage Guide
Use SaaS Mode for panels with 8GB+ RAM. Use Lite Mode for panels with 4GB+ RAM (particularly entry-level hardware configurations or limited processing capability).
2
Install the APK

Open the Downloads folder or File Manager, tap the downloaded APK file, allow unknown sources if prompted, and complete the installation.

3
Grant Required Permissions

Launch the application. Tap Allow when requested for Storage permissions (or toggle "Allow access to manage all files" on Android 11+). The app will reload automatically.

4
Enter School Code

Enter your school's unique license verification key in the School Code field:

SURF-SCHOOL-2026
5
Enter Classroom Name

Enter the Classroom Name exactly as defined in the admin console to load the correct syllabus profile:

Grade-5-A
6
Register Device

Tap the Register button to validate the school profile settings and activate the software license.

7
PIN Verification / First-Time PIN Creation

Access the system using the quick access PIN. Follow the instructions detailed in the PIN Setup & Access section below.

4. PIN Setup & Access Workflow

🔑 Existing Users / Configured Classroom

If the classroom has already completed the initial setup and has an active security PIN:

  • Launch the CePolite application.
  • Enter the pre-configured classroom PIN to access dashboard resources directly.

✨ First-Time Users / New Classroom

If this is the initial launch on a newly registered device:

  1. Log in using the assigned classroom Username and Password.
  2. Upon successful login, you will be prompted to Create a New PIN.
  3. Confirm and save the PIN. Use this PIN for all future quick logins.

6. Troubleshooting Common Installation Issues

Common Issue Action / Resolution
App not installed Uninstall conflicting older versions of CePolite, clear download cache, and retry installer.
Permission denied Go to Android settings > Apps > CePolite > Permissions. Manually grant files/media storage permissions.
Registration failed Verify School Code has no typos. Check the device Wi-Fi connection is active.
PIN not accepted Contact your system administrator to verify details or request a classroom PIN reset.
